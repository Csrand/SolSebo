# SolSebo - Front-end

React SPA para o SolSebo, app de gerenciamento de leitura.

## Setup

```bash
pnpm install
pnpm run dev
```

## Scripts

- `pnpm run dev` — servidor de desenvolvimento (porta 5173)
- `pnpm run build` — compilar para produção
- `pnpm run preview` — preview da build

## Stack

- React 19 + TypeScript
- Vite
- React Router DOM 7
